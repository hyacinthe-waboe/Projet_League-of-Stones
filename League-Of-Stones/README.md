# ⚙️ Backend League of Stones

Serveur pédagogique utilisé par l'interface de **League of Stones**.

![Node.js](https://img.shields.io/badge/Node.js-Server-339933?logo=nodedotjs&logoColor=white)
![Express](https://img.shields.io/badge/Express-API-000000?logo=express)
![MongoDB](https://img.shields.io/badge/MongoDB-Database-47A248?logo=mongodb&logoColor=white)

## 🎯 Rôle

Le backend gère :

- 👤 les utilisateurs et les sessions ;
- 🃏 la collection de cartes ;
- 🔎 le matchmaking ;
- ⚔️ l'état et les actions des parties.

## 🚀 Lancement

Prérequis : Node.js, npm et MongoDB lancé localement.

```bash
npm install
npm start
```

Le serveur écoute sur `http://localhost:3001`.

## ⚠️ À savoir

Ce backend est conservé pour permettre l'exécution locale du projet universitaire. Ses dépendances sont anciennes et il n'est pas conçu pour une mise en production.
