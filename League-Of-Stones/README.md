# Backend League of Stones

Serveur pédagogique Node.js utilisé par le frontend League of Stones. Il gère les utilisateurs, les cartes, le matchmaking et les parties avec Express et MongoDB.

## Prérequis

- Node.js et npm ;
- MongoDB lancé localement.

## Lancement

```bash
npm install
npm start
```

Le serveur écoute sur `http://localhost:3001`.

Ce backend est conservé pour permettre l'exécution locale du projet. Ses dépendances sont anciennes et il n'est pas prévu pour un déploiement en production.
